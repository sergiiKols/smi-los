🔐 АУТЕНТИФИКАЦИЯ

Git запросит учетные данные:

Username: sergiiKols
Password: <ваш Personal Access Token>

🔑 Как получить Personal Access Token:

1. Зайдите на: https://github.com/settings/tokens

2. Нажмите "Generate new token" → "Generate new token (classic)"

3. Заполните:
   • Name: Content Search System
   • Expiration: 90 days (или No expiration)
   • Scopes: ✅ repo (отметьте галочкой - полный доступ к репозиториям)

4. Нажмите "Generate token" внизу страницы

5. ⚠️ ВАЖНО: Скопируйте токен СРАЗУ! Он больше не покажется!

6. Используйте токен как пароль при git push

🚀 СЛЕДУЮЩИЕ ШАГИ (после загрузки на GitHub)

1. Деплой Frontend на Vercel:
   • Зайдите на https://vercel.com
   • New Project → Import Git Repository
   • Выберите sergiiKols/smi
   • Root Directory: web
   • Deploy

2. Деплой Backend на Railway:
   • Зайдите на https://railway.app
   • New Project → Deploy from GitHub repo
   • Выберите sergiiKols/smi
   • Start command: python api_server.py
   • Добавьте environment variables

3. Настройте API ключи (см. SETUP_GUIDE.md)

Подробные инструкции: DEPLOYMENT.md
