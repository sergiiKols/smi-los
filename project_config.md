## Основные услуги компании

### 1. Испытания герметичности систем вентиляции
Проверяем вентиляционные системы на утечки, обеспечивая их эффективность и экономичность.

### 2. Тепловизионное обследование зданий
Диагностируем теплопотери и скрытые проблемы зданий для повышения энергоэффективности.

### 3. Тепловизионное обследование теплотрасс
Обнаруживаем утечки тепла и повреждения в теплотрассах и трубопроводах.

### 4. Проверка герметичности чистых помещений
Контролируем герметичность чистых зон, обеспечивая соблюдение санитарных норм.

### 5. Определение классов герметичности воздуховодов
Тестируем воздуховоды для классификации по герметичности, гарантируя соответствие стандартам.

### 6. Тепловизионное обследование холодильных камер
Выявляем утечки холода и дефекты в холодильных камерах с помощью тепловизоров.

### 7. Измерение воздухопроницаемости зданий
Оцениваем проницаемость воздуха в зданиях, помогая снизить энергозатраты.

### 8. Проверка герметичности при установке газового пожаротушения
Тестируем помещения на герметичность для надежной работы систем газового пожаротушения.

## Технические требования

### API и доступы
- **Gemini API:** Для поиска и анализа статей
- **WordPress REST API:** Для публикации на блоге
- **Facebook Graph API:** Для публикации в Facebook
- **Instagram Graph API:** Для публикации в Instagram

### Расписание
- **Поиск статей:** 1 раз в день
- **Анализ и отбор:** Автоматически через Gemini
- **Публикация:** По расписанию (настраивается)
