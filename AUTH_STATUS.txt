🚀 СЛЕДУЮЩИЕ ШАГИ:

1️⃣  ЗАГРУЗИТЕ НА GITHUB:
   
   git add .
   git commit -m "Add authentication with NextAuth.js"
   git push

2️⃣  НАСТРОЙТЕ VERCEL:

   Settings → Environment Variables → Добавьте:
   
   NEXTAUTH_URL=https://ваш-проект.vercel.app
   NEXTAUTH_SECRET=<сгенерируйте на generate-secret.vercel.app/32>
   ADMIN_USERNAME=admin
   ADMIN_PASSWORD=ваш_надежный_пароль

   Затем Redeploy

3️⃣  ПРОТЕСТИРУЙТЕ:

   • Откройте сайт → перенаправит на /login
   • Войдите с учетными данными
   • Проверьте что все страницы работают
   • Нажмите кнопку "Выход"
