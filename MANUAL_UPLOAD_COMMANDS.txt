🔑 КАК ПОЛУЧИТЬ ТОКЕН:
─────────────────────────────────────────────────────────────

1. Зайдите на: https://github.com/settings/tokens

2. Нажмите "Generate new token" → "Generate new token (classic)"

3. Настройте токен:
   Name: Content Search System
   Expiration: 90 days (или больше)
   Scopes: ✅ repo (отметьте галочкой)

4. Нажмите "Generate token"

5. СКОПИРУЙТЕ ТОКЕН! (он больше не покажется)

6. Используйте токен как пароль при git push
